## Тема 4. Циклы

Три вида циклов в Pascal:

```pascal
// Цикл с параметром
for i := 1 to 10 do
  writeln(i);

// Цикл с предусловием
while k < 11 do begin
  s := s + k;
  k := k + 1;
end;

// Цикл с постусловием
repeat
  s := s + k;
  k := k + 1;
until k > 10;
```

Задание B1: трассировка цикла `while k < 11`. k начинается с 1, s = 0. Цикл выполняется при k = 1, 2, ..., 10. s = 1 + 2 + ... + 10 = 55.

Задание B3, рекурсивная процедура F(n):
```pascal
procedure F(n: integer);
begin
  if n > 0 then begin
    writeln('*');
    F(n-2);
    F(n-1);
    F(n-1);
  end;
  writeln('*');
end;
```
При вызове F(5) считай количество выводов звёздочки. Строй дерево рекурсии вручную.

Задание B4, функция F(x) = 4*(x-5)*(x+3). Программа ищет x в диапазоне [-20, 20], при котором F(x) минимально. Минимум параболы в вершине: x = (5 + (-3)) / 2 = 1. При x=1: F(1) = 4*(-4)*(4) = -64. Ответ: M = 1.

## Тема 8. Рекурсия

Что изучить:
- Базовый и рекурсивный случай.
- Построение дерева вызовов.
- Применение рекурсии для вычисления функций.

Задание A3: F(n) = F(n-1) * F(n-2) + 1, F(0)=1, F(1)=1.
```
F(2) = F(1) * F(0) + 1 = 1*1+1 = 2
F(3) = F(2) * F(1) + 1 = 2*1+1 = 3
F(4) = F(3) * F(2) + 1 = 3*2+1 = 7
F(5) = F(4) * F(3) + 1 = 7*3+1 = 22
F(6) = F(5) * F(4) + 1 = 22*7+1 = 155
```
Правильный ответ: 155.

## Разбор заданий из билета

Задание A5, трассировка с массивом:
```pascal
s := 32;
n := 10;
for i := 0 to n-1 do begin
  s := s + A[i+1] - A[i] + 1
end;
```
Массив возрастающий: A[0] < A[1] < ... < A[10]. Минимальная разность A[i+1] - A[i] = 1 (целые числа). Тогда s = 32 + 10*(1+1) = 32 + 20 = 52. Правильный ответ: 52.

Задание B2, взаимно рекурсивные функции:
F(1)=1, G(1)=1.
F(n) = 3*F(n-1) - 2*G(n-1)
G(n) = F(n-1) + 2*G(n-1)
Считай пошагово:
- n=2: F(2) = 3-2=1, G(2) = 1+2=3
- n=3: F(3) = 3*1-2*3=-3, G(3) = 1+6=7
- n=4: F(4) = 3*(-3)-2*7=-23, G(4) = (-3)+14=11
- n=5: F(5) = 3*(-23)-2*11=-91, G(5) = (-23)+22=-1
G(5) - F(5) = -1 - (-91) = 90.

Задание B5, автомат с четырёхзначным числом:
Правила: перемножить цифры 1*2 и цифры 3*4, записать результаты в порядке убывания.
Пример: 2466 даёт 2*4=8, 6*6=36, результат 368.
Нужно найти наименьшее число, которое даёт 124.
Разбить 124 на два числа в порядке убывания: 12 и 4.
Вариант: a*b=4, c*d=12, то есть 1*4=4 и 3*4=12, число 1434. Или 2*2=4 и 3*4=12, число 2234. Проверяй и ищи минимальное.
